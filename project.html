Data Science Project

Stanley Waldron - 1913361

Welcome to my project for Data Science ECON30006. In this project I aim to investigate whether or not there is a relationship between the features of tracks that have topped the charts in the UK since 1953, and the economic indicators: household consumption and average annual wage growth.

To complete this project, it was necessary for me to familiarise myself with the Spotify API and the ONS API, which would provide almost all of the data I needed. Throughout the project I have worked with Python, JavaScript, HTML and CSS in order to collate and present data in the most effective way, always being conscientious of the replicability of the project. The link to my GitHub can be found back on the homepage (here).
